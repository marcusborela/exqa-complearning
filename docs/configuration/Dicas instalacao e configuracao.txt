Windows
    Baixar versão miniconda python 3.7 em https://docs.conda.io/en/latest/miniconda.html
    Instalar em pasta padrão (C:\Users\borela\Miniconda3)

    Entrar comandos via prompt do anaconda

    conda create -y -n exqa-complearning python=3.7
    # conda remove --name exqa-complearning --all

    Para instalar libraries em um env:
        iniciar anaconda prompt
        executar comandos de conda
            conda env list
            conda activate exqa-complearning


Para configurar vscode para jupyter
    conda activate exqa-complearning
    conda install -n exqa-complearning ipykernel --update-deps --force-reinstall

pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org -r requirements.txt

Para baixar modelo de huggingface:
    git clone https://huggingface.co/unicamp-dl/ptt5-base-en-pt-msmarco-100k-v2

    e mover arquivos para pasta modelo\<nome modelo>

    Precisa instalar git-lfs (Git extension for versioning large files).
    No ubuntu, rodar os comandos abaixo no ubuntu:
        sudo apt install git-lfs
        git lfs install
